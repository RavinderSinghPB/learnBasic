HTML Practice Exercises

Test your HTML knowledge with these exercises.

Exercise 1: Basic Structure

Create a basic HTML structure with a title, heading, paragraph, and an image.

Complete this HTML document:

<!DOCTYPE html>
<html>
<head>
    <title>___________</title>
</head>
<body>
    <_____>My First Webpage</_____>
    <_____>This is a paragraph about my favorite hobby.</_____>
    <_____ src="image.jpg" alt="My hobby">
</body>
</html>

Solution:

<!DOCTYPE html>
<html>
<head>
    <title>My Webpage</title>
</head>
<body>
    <h1>My First Webpage</h1>
    <p>This is a paragraph about my favorite hobby.</p>
    <img src="image.jpg" alt="My hobby">
</body>
</html>

Exercise 2: Links and Lists

Create a navigation menu with links and an unordered list.

Fill in the blanks:

<nav>
    <_____>
        <_____><a _____="index.html">Home</a></_____>
        <_____><a _____="about.html">About</a></_____>
        <_____><a _____="contact.html">Contact</a></_____>
    </_____>
</nav>

Solution:

<nav>
    <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="about.html">About</a></li>
        <li><a href="contact.html">Contact</a></li>
    </ul>
</nav>

Exercise 3: Tables

Create a simple table with headers and data.

Complete this table:

<_____>
    <_____>
        <tr>
            <_____>Name</_____>
            <_____>Age</_____>
            <_____>Country</_____>
        </tr>
    </_____>
    <_____>
        <tr>
            <_____>John</_____>
            <_____>25</_____>
            <_____>USA</_____>
        </tr>
        <tr>
            <_____>Maria</_____>
            <_____>30</_____>
            <_____>Spain</_____>
        </tr>
    </_____>
</_____>

Solution:

<table>
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>Country</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>John</td>
            <td>25</td>
            <td>USA</td>
        </tr>
        <tr>
            <td>Maria</td>
            <td>30</td>
            <td>Spain</td>
        </tr>
    </tbody>
</table>
