Tailwind CSS Basics

Tailwind CSS is a utility-first CSS framework that lets you build designs directly in your markup.

What is Tailwind CSS?

Tailwind CSS is different from frameworks like Bootstrap or Bulma. Instead of pre-designed components, Tailwind provides low-level utility classes that let you build completely custom designs without ever leaving your HTML.

Traditional CSS approach:

/* CSS file */
.btn {
  padding: 0.5rem 1rem;
  border-radius: 0.25rem;
  background-color: #3B82F6;
  color: white;
}

/* HTML file */
<button class="btn">Click me</button>

Tailwind CSS approach:

<button class="px-4 py-2 rounded bg-blue-500 text-white">Click me</button>

With Tailwind, you're writing all your styles directly in your HTML using utility classes. This makes development faster and more intuitive.

Responsive Design

Tailwind makes responsive design simple with breakpoint prefixes:

<div class="w-full md:w-1/2 lg:w-1/3">
  This div is full width on mobile, half width on medium screens,
  and one-third width on large screens.
</div>

Common breakpoint prefixes:

  • sm: - Small screens (640px and up)
  • md: - Medium screens (768px and up)
  • lg: - Large screens (1024px and up)
  • xl: - Extra large screens (1280px and up)
  • 2xl: - 2X large screens (1536px and up)

Step-by-Step Practice Exercises

Let's apply what we've learned with these exercises:

Exercise 1: Create a Button Set

  1. Create a new HTML file with Tailwind CSS included
  2. Create a container div with padding and centering
  3. Add three buttons next to each other:
  4. A primary button (blue background, white text)
  5. A secondary button (gray background, dark text)
  6. A danger button (red background, white text)
  7. Add hover effects to each button
  8. Make the buttons stack vertically on mobile screens

Exercise 2: Profile Card

  1. Create a new HTML file with Tailwind CSS included
  2. Create a card with rounded corners and a shadow
  3. Add a colored header section
  4. Add an avatar image (you can use a placeholder)
  5. Include a name in large, bold text
  6. Add contact information with appropriate icons
  7. Add a footer with social media links

Exercise 3: Responsive Grid Layout

  1. Create a new HTML file with Tailwind CSS included
  2. Create a container with some padding
  3. Add a grid of 6 cards (each with a title, image, and description)
  4. Make the grid show 1 card per row on mobile
  5. Make the grid show 2 cards per row on medium screens
  6. Make the grid show 3 cards per row on large screens
  7. Ensure there's proper spacing between all cards
