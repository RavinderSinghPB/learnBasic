HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

HTML Document Structure

Every HTML document has a basic structure that includes the following elements:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>
  • <!DOCTYPE html> - Declares that this is an HTML5 document
  • <html> - The root element of an HTML page
  • <head> - Contains meta information about the page
  • <title> - Specifies a title for the page
  • <body> - Contains the visible page content

HTML Forms

Forms are used to collect user input:

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name"><br>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email"><br>

  <input type="submit" value="Submit">
</form>

Common input types:

  • text - A single-line text field
  • email - An email field with validation
  • password - A password field that masks input
  • checkbox - A checkbox for multiple selections
  • radio - Radio buttons for single selection
  • submit - A button to submit the form
