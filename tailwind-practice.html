Tailwind CSS Practice Exercises

Test your Tailwind CSS knowledge with these exercises.

Exercise 1: Basic Styling

Apply Tailwind classes to style the following elements:

Add Tailwind classes to create:

<div class="_________">
    <h2 class="_________">Welcome to Tailwind</h2>
    <p class="_________">This is a paragraph with tailwind styling.</p>
    <button class="_________">Click Me</button>
</div>

Requirements:

  • The div should have padding, rounded corners, and a light gray background
  • The heading should be blue, large, and bold
  • The paragraph should have proper spacing and medium gray color
  • The button should be blue with rounded corners, white text, padding, and hover effect

Solution:

<div class="bg-gray-100 p-6 rounded-lg">
    <h2 class="text-blue-600 text-2xl font-bold mb-4">Welcome to Tailwind</h2>
    <p class="text-gray-600 mb-6">This is a paragraph with tailwind styling.</p>
    <button class="bg-blue-500 hover:bg-blue-600 text-white py-2 px-4 rounded">Click Me</button>
</div>

Exercise 2: Responsive Layout

Create a responsive grid layout with Tailwind:

Add Tailwind classes for a responsive 3-column layout that stacks on mobile:

<div class="_________">
    <div class="_________">Column 1</div>
    <div class="_________">Column 2</div>
    <div class="_________">Column 3</div>
</div>

Solution:

<div class="grid grid-cols-1 md:grid-cols-3 gap-4">
    <div class="bg-blue-100 p-4 rounded">Column 1</div>
    <div class="bg-blue-100 p-4 rounded">Column 2</div>
    <div class="bg-blue-100 p-4 rounded">Column 3</div>
</div>

Exercise 3: Card Component

Style a card component with Tailwind CSS:

Add Tailwind classes to create a product card:

<div class="_________">
    <img src="product.jpg" alt="Product" class="_________">
    <div class="_________">
        <h3 class="_________">Product Name</h3>
        <p class="_________">Product description goes here</p>
        <div class="_________">
            <span class="_________">$99.99</span>
            <button class="_________">Add to Cart</button>
        </div>
    </div>
</div>

Solution:

<div class="bg-white rounded-lg shadow-lg overflow-hidden">
    <img src="product.jpg" alt="Product" class="w-full h-48 object-cover">
    <div class="p-4">
        <h3 class="text-xl font-bold text-gray-800 mb-2">Product Name</h3>
        <p class="text-gray-600 mb-4">Product description goes here</p>
        <div class="flex justify-between items-center">
            <span class="text-lg font-bold text-gray-800">$99.99</span>
            <button class="bg-blue-500 hover:bg-blue-600 text-white py-1 px-3 rounded">Add to Cart</button>
        </div>
    </div>
</div>
